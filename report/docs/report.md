# 代码代写

## 项目简介

1. 利用在线 IT 问答网站的提问贴子数据训练模型，实现可以根据内容识别判断编程语言相关 tag
2. 爬取小红书平台代码代写贴子，利用模型对其进行 tag 分析

## Secret of Stackoverflow

为了训练一个可以识别检测 tag 的模型，我们考虑了了多种编程相关网站：

- CSDN
- 知乎
- 微信公众号
- Stackoverflow

由于知乎和微信公众号等平台的训练效果不佳，最终选择了 Stackoverflow 作为我们的训练数据集
